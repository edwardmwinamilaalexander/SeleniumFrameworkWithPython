# Selenium Python Test Automation Framework

A Selenium WebDriver test automation framework built with Python and Pytest. This framework follows the Page Object Model design pattern and includes comprehensive reporting, logging, and parallel testing capabilities.

## üöÄ Features

- **Page Object Model** - Clean separation between test logic and page-specific elements
- **Pytest Integration** - Leverage pytest features for test discovery and execution
- **Allure Reporting** - Generate beautiful, interactive test reports
- **Parallel Testing** - Execute tests in parallel for faster feedback
- **Cross-browser Testing** - Support for Chrome, Firefox, Edge, and Safari
- **Data-driven Testing** - External test data support (JSON/Excel/CSV)
- **Logging Mechanism** - Comprehensive logging for debugging
- **Screenshot Capture** - Automatic screenshot on test failure
- **CI/CD Ready** - GitHub Actions workflow included
- **Environment Management** - Support for multiple environments via .env files
- **Retry Mechanism** - Automatic retry for failed tests
